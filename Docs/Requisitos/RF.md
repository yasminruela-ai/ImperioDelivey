##RF01 
Descrição: O sistema deve permitir o cadastro de usuários utilizando email e senha.

##RF02 
Descrição: O sistema deve permitir autenticação de usuários (login e logout).

##RF03 
Descrição: O sistema deve permitir que o usuário visualize e edite seu perfil.

##RF04 
Descrição: O sistema deve solicitar permissão e capturar a localização do usuário via GPS.

##RF05  
Descrição: O sistema deve validar se o endereço do usuário está dentro da área de entrega.

##RF06 
Descrição: O sistema deve exibir um catálogo de produtos organizado por categorias.

##RF07 
Descrição: O sistema deve permitir visualizar detalhes do produto (nome, preço, descrição e
imagem).

##RF08 
Descrição: O sistema deve permitir adicionar, remover e alterar itens no carrinho.

##RF09 
Descrição:  O sistema deve calcular automaticamente o valor total do pedido.

##RF10
Descrição: O sistema deve permitir a confirmação e envio de pedidos.

##RF11
Descrição: O sistema deve registrar pedidos no banco de dados.

##RF12
Descrição: O sistema deve permitir acompanhar o status do pedido.

##RF13
Descrição: O sistema deve exibir o histórico de pedidos do usuário.

##RF14
Descrição: O sistema deve enviar notificações push sobre atualizações de pedidos.

##RF15 
Descrição: O sistema deve registrar eventos de uso para geração de recomendações.

##RF16
Descrição: O sistema deve gerar sugestões de produtos baseadas no comportamento do
usuário.
