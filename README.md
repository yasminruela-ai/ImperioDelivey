# ImperioDelivey

##  Descrição
Aplicativo desenvolvido para auxiliar no gerenciamento de pedidos de uma hamburgueria, permitindo que clientes visualizem o cardápio e realizem pedidos de forma prática e rápida.

## Problema
Hamburguerias enfrentam dificuldades na organização de pedidos, principalmente em horários de pico. Clientes também podem enfrentar demora no atendimento e dificuldade para visualizar o cardápio completo.

## Público-Alvo
- Clientes da hamburgueria
- Proprietários
- Funcionários responsáveis pelo atendimento e preparo dos pedidos

##  Objetivo
Criar um aplicativo que permita:
- Visualização do cardápio digital
- Realização de pedidos online
- Organização e controle de pedidos
- Melhoria na experiência do cliente
- Integrar IA para recomendações de pedidos

##  Tecnologias Utilizadas
- Figma (protótipo)
- GitHub (versionamento)
- Flutter (FrontEnd)
- NodeJs + Express(Backend)
- Firebase (Database)
