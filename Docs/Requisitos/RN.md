##RN01
Descrição: Pedidos só podem ser realizados se o endereço estiver dentro da área de entrega.

##RN02
Descrição: Apenas usuários autenticados podem realizar pedidos.

##RN03
Descrição: Não é permitido finalizar um pedido com o carrinho vazio.

##RN04
Descrição: O pedido deve seguir a sequência de status: Recebido → Em preparo → Saiu para entrega → Entregue.

##RN05
Descrição: Após o pedido entrar em preparo, ele não poderá ser alterado ou cancelado.

##RN06
Descrição: Toda alteração no status do pedido deve gerar uma notificação para o usuário.

##RN07
Descrição: As recomendações devem considerar histórico de pedidos e frequência de consumo.

##RN08
Descrição: Os dados principais do sistema não devem ser excluídos fisicamente, apenas desativados quando necessário.
