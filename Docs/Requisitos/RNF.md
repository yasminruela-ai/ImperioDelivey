##RNF01
Descrição: O aplicativo deve possuir interface simples, intuitiva e responsiva.

##RNF02
Descrição: O tempo de resposta das operações principais deve ser inferior a 3 segundos.

##RNF03
Descrição: O sistema deve estar disponível 24 horas por dia, salvo manutenção programada.

##RNF04
Descrição: Os dados do usuário devem ser protegidos utilizando mecanismos de segurança.

##RNF05
Descrição: O sistema deve suportar crescimento do número de usuários sem perda significativa de desempenho.

##RNF06
Descrição: O aplicativo deve funcionar em dispositivos Android e iOS.

##RNF07
Descrição: O código do sistema deve seguir boas práticas para facilitar manutenção.

##RNF08
Descrição: O sistema deve tratar erros de forma adequada e apresentar mensagens claras ao usuário.
