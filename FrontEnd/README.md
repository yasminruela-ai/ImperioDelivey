# 🍔 Imperios Burger App

Frontend do aplicativo **Imperios Burger**, desenvolvido em **Flutter** com foco em Android.
Projeto inspirado em apps de delivery como o iFood.

## 🚀 Tecnologias
- Flutter
- Dart
- Material 3

## 🎨 Funcionalidades atuais
- Splash Screen animada
- Tela de Login
- Tela de Cadastro
- Home com UI premium
- Animações suaves
- Identidade visual
- Recursos nativos Android

## 📱 Status do projeto
🚧 Em desenvolvimento (frontend apenas)
